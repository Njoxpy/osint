## Usage Guidelines

### Educational Purpose
These images are provided for educational and training purposes only. When using these materials:

- Respect copyright and intellectual property rights
- Use only for legitimate OSINT training
- Do not use for malicious purposes
- Follow ethical guidelines

### Analysis Practice

Use these images to practice:

1. **EXIF Data Extraction**
   ```bash
   exiftool images/sample.jpg
   ```

2. **Reverse Image Searching**
   - Upload to Yandex Images
   - Search on Google Images
   - Check TinEye for matches

3. **Metadata Analysis**
   - Check for GPS coordinates
   - Analyze timestamp data
   - Verify camera information

4. **Visual Analysis**
   - Identify landmarks and locations
   - Analyze environmental indicators
   - Look for cultural and social clues

### Privacy Considerations

- All images in this gallery are either public domain or used with permission
- No private or sensitive information is exposed
- Images have been reviewed for EXIF data before inclusion
- Use responsibly and ethically

## Contributing Images
